# Задание: Семантическая разметка 

## Шаг 1. Создание проекта

1. Создайте новый репозиторий под названием `mobdev-lab-git-ваша_фамилия`.
2. Создайте ветку `done` и запушьте её (держите её пустой, так как туда вы будете делать pull request)
3. Создайте файл `index.html`.
4. Сделайте шаблон HTML (документ с `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
5. Добавьте заголовок первого уровня `<h1>` с текстом:

   ```
   Это мое приложение
   ```

6. Сохраните изменения.
7. Зафиксируйте изменения в истории проекта (сделайте первый коммит).

---

## Шаг 2. Добавление футера

1. В `index.html` внизу страницы добавьте футер `<footer>` с текстом:

   ```
   Фамилия Имя 2026
   ```

2. Сохраните изменения.
3. Зафиксируйте изменения в истории проекта (сделайте второй коммит).

---

## Шаг 3. Создание новой ветки

1. Создайте новую ветку с названием `feature/content`.
2. В этой ветке добавьте параграф `<p>` с текстом:

   ```
   Это тестовый контент для страницы.
   ```

3. Сохраните изменения и зафиксируйте их (сделайте коммит).

---

## Шаг 4. Работа с ветками и мердж

1. Вернитесь в основную ветку проекта (например, `main`).
2. Измените заголовок `<h1>` на:

   ```
   Мое приложение на Git
   ```

3. Сохраните изменения и сделайте коммит.
4. Объедините ветку `feature/content` с основной веткой, сохранив все изменения обеих веток.

---

## Шаг 5. Отмена изменений (revert)

1. В основной ветке удалите футер `<footer>`.
2. Сохраните изменения.
3. Создайте коммит, который **отменяет удаление футера**, чтобы футер снова появился.
- Когда делаете revert до коммита у вас возникает merge conflict, закройте его с такими изменениями, чтобы footer вернулся
- Добавьте решение конфлика `git add .`
- Продолжите revert `git revert --continue`
4. История должна показывать все изменения: добавление футера, удаление и восстановление.

---

## Шаг 6. Финальная проверка

1. Проверьте, что в основной ветке есть:

   * Заголовок `<h1>`: `Мое приложение на Git`
   * Параграф `<p>` из ветки `feature/content`
   * Футер `<footer>` с вашим именем и годом
